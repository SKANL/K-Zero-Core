# Filosofía de K-Zero-Core

K-Zero-Core es un asistente operativo local-first y gratis por defecto. Su lugar no es competir frontalmente con agentes de código de Big Tech, sino ayudar a usuarios con bajo presupuesto a convertir voz, archivos, documentos y conocimiento local en resultados útiles.

## Principios

- Gratis primero: siempre debe existir una ruta usable con Ollama, herramientas locales y proveedores gratuitos.
- Cloud opcional: OpenAI-compatible, APIs de búsqueda y servicios pagos enriquecen el resultado solo cuando el usuario los configura.
- Privacidad visible: cada workflow declara si es local, usa red o envía datos a cloud.
- Resultados antes que arquitectura: usuarios no técnicos eligen tareas; usuarios técnicos pueden editar workflows, toolsets y providers.
- Escritura visible: los entregables se crean como copias en `exports` y otras escrituras locales, como transcripciones o índices RAG, se declaran antes de ejecutar.

## Diferenciador

K-Zero-Core prioriza español, voz, documentos reales, RAG local, memoria curada, archivos locales y entregables nativos. Los agentes de código pueden ser mejores en repositorios y terminal; K-Zero-Core debe ser más cómodo para trabajo personal cotidiano.
